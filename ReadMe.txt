Learning Needs Analysis

coresurvey

This code is an an activity module (mod) in the process of an upgrade from Moodle 1.9 to Moodle 2.3.

Its function is to carry out a reflective survey:
 - to choose an appropriate role by analysing activities you like,
 - to assess your competence to carry that role out 
 - to offer relevant guidance for learning to improve competence in that role.
 
First developed by Core Education in New Zealand for Macmillan Cancer Support,
it is now being developed further by Richard Millwood at Core Education UK.
