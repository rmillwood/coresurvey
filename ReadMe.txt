Learning Needs Analysis

 'coresurvey' plug-in for Moodle
 Core Education UK
 http://www.core-ed.org.uk
 Author: Richard Millwood, based on code by Nigel Hulls of CORE Education NZ
 E-mail: richard.millwood2core-ed.org.uk
 
This code is an an activity module (mod) for Moodle 2.

Its function is to carry out a reflective survey:
 - to choose an appropriate role by analysing activities you like,
 - to assess your competence to carry that role out 
 - to offer relevant guidance for learning to improve competence in that role.
 
The development has been funded by Macmillan Cancer Support and used on their Learn Zone.
